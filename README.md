## About This Repository

I come from a non-tech background and started learning Python from scratch.
This repository is proof that concepts only make sense when you build with them.

Every project here is built using only the Python concepts I had learned up to that point —
no shortcuts, no skipping ahead. Each one was a test of whether I actually understood
what I studied, or just watched tutorials.

My goal is simple — show that Python fundamentals are not just theory.
Without a strong foundation in lists, loops, dictionaries, conditionals and functions,
every bigger project you build later will have cracks in it.
These mini projects are how I made sure mine didn't.

## Mini Projects

### 01 — Student Grade Tracker
📁 `grade-tracker/grade_tracker.py`

**What it does:**
- Stores a list of students with their scores
- Takes a new student's name and score as user input
- Assigns a grade (A / B / C / F) to every student based on their score
- Finds and displays the topper automatically

**Concepts used:**
`lists` `dictionaries` `for loop` `conditionals` `input` `type casting` `variables`

**How to run:**
```bash
cd grade-tracker
python grade_tracker.py
```

---

### 02 — Bollywood Quiz Game
📁 `quiz-game/quiz_game.py`

**What it does:**
- Asks 5 Bollywood questions one by one
- Takes user answer as input for each question
- Checks if the answer is correct or wrong
- Shows final score out of 5 at the end

**Concepts used:**
`lists` `dictionaries` `for loop` `conditionals` `input` `comparison operators` `score counter`

**How to run:**
```bash
cd quiz-game
python quiz_game.py
```
